# research-toolkit-kidney-workshop# Research Toolkit — Surgical Anatomy of the Kidney

An interactive, mobile-first web tool designed for medical students participating in the "Surgical Anatomy of the Kidney" workshop at the Department of Anatomy and Embryology, "Grigore T. Popa" University of Medicine and Pharmacy, Iași, Romania.

## What It Does

This single-page application serves as a comprehensive research companion that helps medical students navigate scientific literature and AI-powered tools during anatomy workshops. Students access it on their phones via QR code and can immediately search, explore, and learn.

## Features

**Multi-Engine Search** — One search box, four destinations. Students type a query (e.g., "renal artery anatomical variation") and search directly on PubMed, Google Scholar, Semantic Scholar, or Consensus with a single tap. Pre-loaded quick tags cover key renal anatomy topics: kidney CBCT, renal artery CT, transplant anatomy, horseshoe kidney, renal vein variations, and 3D printing in surgical planning.

**AI Research Tools Directory** — Curated collection of free AI-powered research tools with descriptions and use cases: Consensus (evidence-based answers), Elicit (automated data extraction), Semantic Scholar (200M+ papers with AI summaries), scite.ai (citation context analysis), Claude/ChatGPT (summarization and explanation), and Perplexity AI (cited web search).

**Visual Anatomy Resources** — Direct links to interactive 3D anatomy platforms (Complete Anatomy, Radiopaedia, Kenhub, IMAIOS e-Anatomy) focused on renal structures.

**Step-by-Step Research Guide** — Six-step methodology from clinical question formulation through MeSH term construction, intelligent filtering, AI-assisted summarization, anatomy-to-pathology workflow, and source verification. Each step includes copy-paste-ready AI prompts.

**Ready-Made Prompts** — Five kidney-specific prompts covering surgical anatomy, vascular variations, pathology correlation, PubMed search optimization, and mini-review structuring. Tap any prompt to copy it.

**Complete Research Workflow** — Visual 8-step pipeline from question to written output, integrating all tools.

**AI Ethics Section** — Clear do's and don'ts for responsible AI use in medical research.

## Tech Stack

Pure HTML/CSS/JavaScript. No frameworks, no dependencies, no build step. Single file deployment. Dark mode UI optimized for mobile screens.

## Usage

Host on GitHub Pages, share the link or generate a QR code. Students scan and start researching immediately during workshops.
